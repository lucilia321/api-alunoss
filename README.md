# API de Alunoss - Flask

## Descrição
Esta é uma API simples para gerenciamento de alunos, utilizando Python e Flask.  
Os dados são armazenados em um arquivo JSON (`alunoss.json`).  
A API implementa todas as operações básicas de CRUD: Create, Read, Update e Delete.
